# React学習計画

## 学習の進め方

### フェーズ1：写経（サンプルコードを書き写す）
1. サンプルコードを提示してもらう
2. 自分でコードを書き写す
3. 動作確認
4. **コードの全ての部分を理解するまで質問を繰り返す**
5. 学習記録をREADME.mdに追記
6. Gitにコミット・push

### フェーズ2：実装（お題から自分で作る）
1. お題だけを受け取る（ヒントなし）
2. 自分で調べながら実装
3. 動作確認
4. レビュー・改善
5. 学習記録をREADME.mdに追記
6. Gitにコミット・push

### フェーズ3：応用（複雑な機能）
- フェーズ1とフェーズ2を組み合わせ
- より実践的なアプリ開発
