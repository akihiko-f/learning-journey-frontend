## 学習メモ

### setIntervalの基本

```jsx
useEffect(() => {
  let interval = null

  if (isRunning && timeLeft > 0) {
    interval = setInterval(() => {
      setTimeLeft(prev => prev - 1)
    }, 1000)
  }

  return () => clearInterval(interval) // クリーンアップ
}, [isRunning, timeLeft])
```

ポイント：
- useEffect内でsetIntervalを使う
- returnでclearIntervalを呼ぶ（メモリリーク防止）
- 依存配列に必要な値を入れる
