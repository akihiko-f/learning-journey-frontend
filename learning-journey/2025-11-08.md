## 学んだこと

### 1. textareaの使い方
- 複数行入力が必要な場合は`<textarea>`要素を使用
- `rows`属性で初期の行数を指定可能
- `resize: vertical`でCSSで垂直方向のみリサイズ可能にする

### 2. 検索機能の実装
- `Array.filter()`と`String.includes()`を組み合わせて検索を実現
- `toLowerCase()`で大文字小文字を区別しない検索を実装
- フィルタリングされた配列を使用することで、元のデータは変更せずに表示だけを変更

### 3. CSSによるスタイリング
- `box-sizing: border-box`でpaddingを含めた幅計算
- `border-radius`で角を丸く、`box-shadow`で立体感を表現
- `:hover`疑似クラスでインタラクティブな効果を追加
- `white-space: pre-wrap`で改行を保持して表示

## 学習メモ

### 検索フィルタリングの実装
```jsx
const filteredMemos = memos.filter(memo => {
  const searchLower = searchTerm.toLowerCase()
  return memo.title.toLowerCase().includes(searchLower) ||
         memo.content.toLowerCase().includes(searchLower)
})
```

- `filter()`で条件に合う要素のみを抽出
- `includes()`で部分一致検索
- タイトルと本文の両方から検索可能

### CSSのポイント
```css
.memo-form button:hover {
  background-color: #45a049;
}
```

- `:hover`でマウスオーバー時の見た目を変更
- `transition`プロパティでなめらかな変化を実現（今回は未使用だが追加可能）
