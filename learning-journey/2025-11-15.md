## 学習メモ

### テーブルのスタイリング手法

Reactでテーブルを使う際のスタイリングのポイント：

```jsx
<table style={{
    width: '100%',
    maxWidth: '600px',
    borderCollapse: 'collapse'
}}>
    <tbody>
        <tr style={{ borderBottom: '1px solid #ddd' }}>
            <td style={{ padding: '0.75rem', fontWeight: 'bold', width: '150px' }}>
                会社名
            </td>
            <td style={{ padding: '0.75rem' }}>
                株式会社サンプル
            </td>
        </tr>
    </tbody>
</table>
```

**ポイント**：
- `borderCollapse: 'collapse'`：セル間の隙間をなくす
- 各行に`borderBottom`を設定して区切り線を表現
- 左カラムに固定幅を設定して整列
- `padding`で適切な余白を確保

### CSSグラデーションの活用

収入・支出・残高カードでグラデーションを使用：

```css
.summary-item.income {
  background: linear-gradient(135deg, #51cf66, #37b24d);
  color: white;
}

.summary-item.expense {
  background: linear-gradient(135deg, #ff6b6b, #ee5a6f);
  color: white;
}

.summary-item.balance {
  background: linear-gradient(135deg, #4dabf7, #228be6);
  color: white;
}
```

**グラデーション角度の意味**：
- `135deg`：左上から右下へのグラデーション
- `180deg`：上から下へのグラデーション（棒グラフで使用）
- `90deg`：左から右へのグラデーション（水平棒グラフで使用）

### 棒グラフの実装パターン

**縦棒グラフ（収入vs支出）**：
```css
.comparison-chart {
  display: flex;
  justify-content: center;
  align-items: flex-end;  /* 下揃え */
  gap: 40px;
  height: 250px;
}

.bar {
  width: 80px;
  min-height: 20px;
  border-radius: 8px 8px 0 0;  /* 上部のみ角丸 */
}
```

**水平棒グラフ（カテゴリー別）**：
```css
.chart-bar-container {
  flex: 1;
  height: 30px;
  background-color: #f0f0f0;
  border-radius: 4px;
  overflow: hidden;
}

.chart-bar {
  height: 100%;
  background: linear-gradient(90deg, #ff6b6b, #ee5a6f);
  transition: width 0.3s ease;  /* アニメーション */
}
```

### Flexboxレイアウトの実践的な使い方

サマリーカードの3カラムレイアウト：
```css
.summary {
  display: flex;
  gap: 15px;  /* カード間の隙間 */
  margin-bottom: 30px;
}

.summary-item {
  flex: 1;  /* 均等に幅を分配 */
  padding: 20px;
  border-radius: 8px;
}
```

**Flexboxのプロパティ**：
- `display: flex`：Flexコンテナを作成
- `flex: 1`：子要素が均等に伸縮
- `gap`：子要素間の隙間を一括設定
- `align-items: flex-end`：下揃え（棒グラフで重要）

### トランザクション項目のデザインパターン

```css
.transaction-item {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 15px;
  border-left: 4px solid;  /* 左側にアクセント線 */
}

.transaction-item.income {
  background-color: #d3f9d8;
  border-color: #51cf66;
}
```

**デザインのポイント**：
- `border-left`で左側にアクセントカラー
- 収入と支出で背景色を変えて視認性を向上
- `justify-content: space-between`で左右に要素を配置
